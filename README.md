# Laravel + Vue: Интеграция с Яндекс Картами

## Описание проекта

Проект реализует веб-приложение на **Laravel + Vue.js**, позволяющее:

- Авторизоваться через логин/пароль.
- Настроить ссылку на Яндекс.Карты через страницу настроек.
- Выводить все отзывы с карточки компании Яндекса.
- Отображать рейтинг компании и общее количество отзывов.
- Использовать постраничную навигацию и сортировку отзывов по новизне.

> ⚠️ Важно: **для запуска проекта требуется Docker и Docker Compose**.  
> Без Docker приложение не будет работать, так как все сервисы (PHP, MySQL, Node.js, Nginx) запускаются в контейнерах.

Проект полностью развёрнут через **Docker**, что упрощает установку и запуск.

## Стек технологий

- **Backend:** Laravel 12  
- **Frontend:** Vue 3 + Vite  
- **База данных:** MySQL 8  
- **Веб-сервер:** Nginx  
- **Контейнеризация:** Docker + Docker Compose

## Установка и запуск

1. **Клонируем репозиторий:**

```bash
git clone https://github.com/vladislavkip/Daily-Grow.git
```

2. **Создаём и запускаем контейнеры:**

```bash
docker-compose up -d --build
```

⚠️ При первом запуске также необходимо выполнить миграцию БД
```bash
docker-compose exec app php artisan migrate
```

## Доступ к приложению:

Laravel + Vue: http://localhost:8000

Vite Dev Server: http://localhost:5173

## Использование

**Авторизация:**

Введите логин/пароль, чтобы войти в систему.

**Страница настроек:**

Вставьте ссылку на карточку компании Яндекса.

Сохраните настройки.

**Отзывы:**

Просмотр всех отзывов компании.

Отображается рейтинг и общее количество отзывов.

Доступна постраничная навигация.

| Сервис | Описание |
|--------|----------|
| app    | PHP + Laravel |
| nginx  | Веб-сервер Nginx |
| db     | MySQL 8 |
| node   | Node.js + сборка Vue |

## Порты:

Laravel: http://localhost:8000

Vite Dev Server: http://localhost:5173

MySQL: 3306

## Полезные команды
# Остановка контейнеров
```bash
docker-compose down
```
# Перезапуск контейнеров
```bash
docker-compose up -d
```
# Тесты Laravel
```bash
docker-compose exec app php artisan test
```
